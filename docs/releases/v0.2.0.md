# bhpan-cli v0.2.0 Release Notes

发布日期：2026-03-18

## 新增功能

### 递归列出目录 (ls -R)

- `bhpan ls /home -R` 递归列出所有文件和目录
- `bhpan ls /home -R -L 2` 限制递归深度
- 输出格式：显示完整逻辑路径（如 `/home/code/project/file.txt`）
- 与 Linux `ls -R` 类似，但优化为每行一条记录便于脚本处理

### 正则表达式过滤 (--regex)

- `bhpan ls /home -R --regex '.*\\.pdf$'` 过滤 PDF 文件
- `bhpan tree /home -L 4 --regex '.*report.*'` 过滤并显示上下文
- 支持 JavaScript 正则表达式语法
- 匹配目标为完整逻辑路径

### 命令行参数改进

- 所有命令现在支持标志位前后位置无关（如 `ls -R /path` 和 `ls /path -R` 都有效）
- 修复了 `head -n 5 file` 参数顺序问题
- 修复了 `rm -r /path` 参数顺序问题

## 改进

- 深度优先遍历算法优化
- 更清晰的错误提示信息
- 类型安全保证（通过 TypeScript 严格检查）

## 兼容性

- 向后兼容 v0.1.x 所有功能
- Node.js 版本要求：>= 22
