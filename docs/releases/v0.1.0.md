# bhpan-cli v0.1.0

首个可发布版本。

## Highlights

- 提供单次 CLI 与交互式 shell 两种使用方式
- 完成针对北航当前线上站点的 OAuth2 / Hydra 登录适配
- 支持 `/home` 路径别名与共享核心路径解析
- 支持基础文件管理：
  - `ls`
  - `tree`
  - `stat`
  - `mkdir`
  - `rm`
  - `mv`
  - `cp`
  - `cat`
  - `head`
  - `tail`
  - `touch`
  - `upload`
  - `download`
- 支持外链管理：
  - 匿名分享
  - 实名分享
  - 多分享并存时的查看与删除
- 发布为标准 npm CLI 包，安装后直接提供 `bhpan` 命令

## Verified

已使用真实北航账号验证：

- 登录与 token 刷新
- `/home` 目录解析
- `ls /home`
- 小文件上传下载
- `tree -L/--depth`、排序
- 匿名分享 create/show/delete
- 实名分享 create/show/delete
- 同一路径下匿名 + 实名分享并存时的 show/delete

## Known Gaps

- `mv` / `cp` 仍需要补完整的真实账号专项回归记录
- 大文件与递归目录传输还需要更系统的联调
- 中断恢复与更细粒度的传输异常处理仍待增强
- shell 还未加入命令补全

## Install

```bash
npm install -g bhpan-cli
```
